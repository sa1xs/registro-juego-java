# Registro de Juego en Java

Proyecto en Java desarrollado en consola para registrar jugadores, cantidad de partidas y puntajes obtenidos.

## Descripción
El programa permite:
- Registrar la cantidad de jugadores
- Ingresar el nombre de cada jugador
- Registrar el número de partidas jugadas
- Guardar los puntajes de cada partida
- Mostrar resultados individuales y globales

## Funcionalidades
- Puntaje máximo por jugador
- Puntaje mínimo por jugador
- Promedio de puntajes
- Cantidad de partidas con puntaje mayor a 1000
- Jugador con el puntaje más alto
- Jugador con el mejor promedio

## Tecnologías usadas
- Java
- Arreglos
- Matrices
- Ciclos
- Condicionales
- Entrada y salida por consola
